# 找规律类型的

**1. 一个数分成几份，可以被 3 整除的最大份数。比如 12345 分成12 3 45 结果为3.**

该题基于一个事实：能被3整除的数的各位数字相加也可被3整除，所以可以利用余数，余数分别出现过1、2，则当前遍历的组合必有一个子组合可被3整除。

还有，我刚开始理解是，如何划分这个数，使得每一份都可以被3整除，并且分数最大。后来看人家代码发现并不需要每一份都被3整除。。。那有啥意义。。。
